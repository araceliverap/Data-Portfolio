# Análisis de Datos y Visualización de Tendencias

## Contexto del proyecto
Durante el bootcamp de TripleTen, trabajé con datasets simulados de ventas y clientes para identificar tendencias clave, patrones de comportamiento y oportunidades de optimización en la toma de decisiones.

## Mi análisis
- Limpié y transformé datos de múltiples fuentes usando Python y Pandas, resolviendo inconsistencias y valores faltantes.  
- Realicé análisis exploratorio para identificar tendencias y patrones relevantes.  
- Construí visualizaciones usando Matplotlib y Seaborn para comunicar insights a stakeholders.

## Conclusiones principales
- Identifiqué segmentos de clientes con mayor recurrencia de compra.  
- Detecté patrones estacionales de ventas que permitieron sugerir estrategias de promoción.  
- Generé dashboards visuales que facilitaron la toma de decisiones estratégicas.
